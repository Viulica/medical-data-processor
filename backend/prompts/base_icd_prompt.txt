════════════════════════════════════════════════════════════════════════════════

RULE 1: DOCUMENTATION-FIRST CODING - VISUAL INDICATORS MATTER
- ONLY code diagnoses that are EXPLICITLY DOCUMENTED in the medical record
- Do NOT infer diagnoses from general medical history
- Do NOT assume a comorbidity applies just because it's listed in "Past Medical History"
- ⚠️ CRITICAL: When diagnoses are listed but NOT circled, marked, or highlighted:
  * Only code the diagnoses that ARE marked/circled/highlighted
  * Unmarked diagnoses are template items, not confirmed present diagnoses
  * Exception: The circled/marked ones are the ACTUAL diagnoses present in the patient
- ⚠️ CRITICAL: When many diagnoses are listed but ONLY SOME are in RED COLOR:
  * Only code the diagnoses shown in RED color
  * Red color indicates the diagnoses that are present in the patient
  * Diagnoses in normal/black text are part of a checklist, not confirmed present
- Check the PROCEDURE section and POST-OP DIAGNOSIS first
- Match the documented indication to the procedure being performed
- If a diagnosis is not explicitly documented or visually marked/colored in this encounter, do NOT code it

RULE 2: NEVER CODE BOTH SYMPTOM AND CONDITION
- NEVER code both R-codes (symptoms) AND their related confirmed condition
- Example: If record says "abdominal pain, found to have K63.5 polyp" → code K63.5 ONLY, not both R10.11 and K63.5
- Example: If "dyspepsia diagnosed as K29.70 gastritis" → code K29.70 ONLY, not both R10.13 and K29.70
- Symptoms are supporting evidence for diagnosis, not separate codes to bill
- Always use the CONFIRMED condition, never the symptom alone

RULE 3: ACTIVE CODE VALIDATION - REJECT THESE INACTIVE/RETIRED CODES
- NEVER use these inactive/retired codes:
  * Z86.010 (INACTIVE) → Replace with Z86.0100 (Personal history of colonic polyps)
  * Z53.3 (INACTIVE) → Do NOT use, find the actual diagnosis instead
  * M54.5 (INACTIVE/AMBIGUOUS) → Use M54.89 (other specified dorsalgia) if applicable
- ONLY use codes valid and active as of December 2025
- If uncertain whether a code is active, LEAVE THE FIELD EMPTY rather than guess
- When in doubt about code status, err on the side of being conservative

RULE 4: ICD1 MUST BE PROCEDURE-RELATED OR INDICATION-BASED, NEVER A COMORBIDITY
- ICD1 is the PRIMARY diagnosis for THIS procedure, not a comorbid condition
- Match the procedure to its documented indication FIRST
- Examples:
  * Colonoscopy → ICD1 should be Z12.11 (screening) or the finding/symptom (K63.5, K21.9, etc), NOT I10 hypertension
  * Cataract surgery → ICD1 should be H25.xxx (cataract), NOT H40.x (glaucoma) unless glaucoma is the PRIMARY reason
  * EGD → ICD1 should be upper GI indication (K29.70, K44.9, K21.9), NOT comorbidity
  * Knee arthroplasty → ICD1 should be M17.xx (osteoarthritis), NOT I10 or E11.9
- Comorbidities go in ICD2-4 ONLY (and only if ASA 3+ AND documented AND relevant)
- If no procedure-related diagnosis is found, leave ICD1 empty rather than use a comorbidity

RULE 5: SCREENING vs HISTORY CODE PRIORITY
- Z12.x codes (Encounter for screening) have PRIORITY over Z86.x codes (Personal history)
- NEVER use Z86.010 (Personal history of polyps) as ICD1 when Z12.11 (Screening for colorectal cancer) is appropriate
- When procedure is for SCREENING → Z12.11 goes in ICD1
- When patient has HISTORY of polyps but THIS procedure is SCREENING → Z12.11 in ICD1, Z86.x in ICD2
- History codes (Z86.x) are SECONDARY at best, never PRIMARY
- Screening/encounter codes describe the PURPOSE of THIS encounter, history codes describe PAST encounters

RULE 6: MATCH COMORBIDITIES TO ASA STATUS ONLY
- ASA 1-2: Do NOT add ANY comorbidities from medical history → code ONLY procedure-related diagnosis
- ASA 3+: MAY add 1-2 most relevant comorbidities IF documented AND relevant to anesthesia risk
- Do NOT fill ICD slots just because space is available
- Comorbidities must be EXPLICITLY DOCUMENTED in the current record, not assumed from history

4. CRITICAL COLONOSCOPY ICD CODING (FACILITY GOLDEN STANDARD - NO EXCEPTIONS):
════════════════════════════════════════════════════════════════════════════════

SCENARIO 1: Screening Colonoscopy WITH Polyp Removal (Polypectomy found/performed in THIS procedure)
  • ICD1: Z12.11 (Encounter for screening for malignant neoplasm of colon)
  • ICD2: K63.5 (Polyp of colon) [or D12.x site-specific if location documented]
  • ICD3: Leave empty unless ASA 3+ with relevant comorbidity
  • ICD4: Leave empty unless ASA 3+ with relevant comorbidity
  • KEY: "Polypectomy" MUST be explicitly documented in post-op findings

SCENARIO 2: Screening Colonoscopy WITH History of Polyp (NO new polyps found/removed THIS procedure)
  • ICD1: Z12.11 (Encounter for screening for malignant neoplasm of colon)
  • ICD2: Z86.0100 (Personal history of polyps of digestive tract) — NOT Z86.010 (INACTIVE)
  • ICD3: Leave empty unless ASA 3+ with relevant comorbidity
  • ICD4: Leave empty unless ASA 3+ with relevant comorbidity
  • KEY: History code goes in ICD2, NOT ICD1

SCENARIO 3: General screening colonoscopy with NO findings
  • ICD1: Z12.11 (Encounter for screening for malignant neoplasm of colon)
  • ICD2: Leave empty unless family history (Z80.0) or personal history (Z86.0100) explicitly documented
  • ICD3: Leave empty
  • ICD4: Leave empty

SCENARIO 4: Surveillance Colonoscopy (history of polyps, positive Cologuard, or follow-up — NO symptoms)
  • ICD1: Z12.11 (Encounter for screening for malignant neoplasm of colon)
  • ICD2: K63.5 if new polyps found/removed, OR Z86.0100 if no new findings
  • ICD3: Additional history codes or comorbidities if ASA 3+
  • ICD4: Leave empty unless ASA 3+ with relevant comorbidity
  • KEY: Surveillance is still screening intent — Z12.11 is PRIMARY. Includes "Hx of polyps", "positive Cologuard/FIT", "surveillance colonoscopy"
  • KEY: Only when SYMPTOMS are documented (pain, bleeding, etc) does it become diagnostic (Scenario 5)

SCENARIO 5: Diagnostic Colonoscopy (symptom-based — pain, bleeding, bowel changes)
  • ICD1: [Symptom/finding code] (e.g., K62.5 rectal bleeding, R10.13 abdominal pain, R19.4 bowel changes)
  • ICD2: Z12.11 only if screening was ALSO part of the documented procedure reason
  • ICD3: History codes (Z80.0, Z86.0100) or supportive findings if documented
  • ICD4: Additional comorbidities if ASA 3+ AND documented
  • KEY: Symptoms MUST be explicitly documented as the reason for the procedure

KEY RULES:
  • Screening AND surveillance colonoscopy = Z12.11 is PRIMARY (ICD1)
  • Only symptom-based/diagnostic colonoscopy uses the symptom as ICD1
  • If polyps found and removed = K63.5 goes in ICD2
  • If only history of polyps (no NEW findings) = Z86.0100 goes in ICD2 (NOT Z86.010 which is INACTIVE)
  • NEVER use both K63.5 AND Z86.0100 in same case (one or the other, not both)
  • NEVER use Z12.11 for symptom-based colonoscopy as ICD1 (only as secondary if screening was also done)
  • Post-op diagnosis ALWAYS takes priority
  • Do NOT fill remaining ICD slots with filler codes (obesity E66.9, anxiety F41.9, etc) - leave empty if not relevant

   - EXCEPTION: If the record documents complications such as tobacco/substance use in pregnancy (O99.214, O99.314), preeclampsia (O14.x), fetal distress (O77.x), gestational diabetes (O24.x), or other obstetric complications, use the complication code as ICD1 instead of O80.

9. CRITICAL ENCOUNTER TYPE RULE (7th character):
════════════════════════════════════════════════════════════════════════════════
   - For injury/fracture codes (S-codes, T-codes): Use "A" (initial encounter) unless the document explicitly states this is a follow-up or subsequent visit for a previously treated injury. Anesthesia for surgery is almost always an initial encounter (A), NOT subsequent (D) or sequela (S).
   - Example: S42.391A (initial) NOT S42.391D (subsequent).

10. CRITICAL LATERALITY AND SPECIFICITY RULE:
════════════════════════════════════════════════════════════════════════════════
   - Carefully read the surgical site (left, right, bilateral) from the operative note and match the ICD code laterality accordingly.
   - When the document does not specify laterality, use the "unspecified" variant of the code (typically ending in 9). Do NOT guess laterality.
   - Always prefer the most specific ICD code supported by the documentation. For example, if the document says "open-angle glaucoma of left eye," use H40.1122 (specific) rather than H40.9 (unspecified).
   - ⚠️ POST-OP DIAGNOSIS ALWAYS TAKES PRIORITY: Always read and use the Post-Operative Diagnosis for ICD1, NOT the Pre-Op Diagnosis. The post-op is what actually happened during the procedure and is the definitive diagnosis for coding. If the post-op diagnosis is different from or more specific than the pre-op (e.g., pre-op says "gallstones" K80.00 but post-op says "acute cholecystitis" K81.0, or pre-op says "osteomyelitis" but post-op says "diabetic complication"), use the post-op. Only fall back to pre-op if no post-op diagnosis is documented.
