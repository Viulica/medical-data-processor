RULES FOR PICKING the cpt code and modifiers (LT = left, RT = right, 50 = bilateral)

⚠️ GENERAL DISAMBIGUATION RULE — USE THE SURGICAL SITE AND LATERALITY ⚠️

Charts often record blocks vaguely ("brachial plexus block", "TAP block", "spinal", "peripheral nerve block", "regional") without specifying the exact subtype, laterality, or single-shot vs catheter. When the chart is ambiguous, you MUST disambiguate by cross-referencing TWO things:

1. THE SURGICAL SITE / MAIN ANESTHESIA CPT — what part of the body was operated on. The block must anatomically cover that site.
2. THE LATERALITY OF THE SURGERY — unilateral surgery never gets a bilateral block; bilateral surgery rarely gets a unilateral block.

PROCESS for every block you find:
- Step A: Identify the block FAMILY (brachial plexus, TAP, paravertebral, neuraxial, vascular line, etc.).
- Step B: Determine the SURGICAL SITE from the procedure description and the main anesthesia CPT.
- Step C: Determine LATERALITY (LT / RT / bilateral) from the procedure description, "Site:", "Laterality:", or the diagnosis (e.g. "right inguinal hernia" → RT).
- Step D: Determine SINGLE-SHOT vs CONTINUOUS CATHETER. If the chart only says "block" with no catheter/infusion mention, default to SINGLE-SHOT.
- Step E: Pick the most specific CPT that matches site + laterality + technique. NEVER default to the most common CPT in the family if the surgical site/laterality contradicts it.

KEY CROSS-CHECKS (apply these whenever the chart is vague):

A. BRACHIAL PLEXUS FAMILY — the subtype depends on the SURGICAL SITE:
   - SHOULDER surgery (rotator cuff, shoulder arthroscopy, total shoulder, clavicle) → INTERSCALENE → 64415 (single) / 64416 (catheter)
   - ELBOW / FOREARM / WRIST / HAND surgery (cubital tunnel, carpal tunnel, distal radius, hand mass, fistula creation) → AXILLARY → 64417 (single) / 64418 (catheter)
   - SUPRACLAVICULAR is sometimes used for upper-arm/elbow — if the chart explicitly says "supraclavicular" use the same code as interscalene for our purposes (64415).
   - If the chart only says "brachial plexus block" and the case is anesthesia CPT 01710/01716/01740/01758/01760/01810/01820/01830/01832/01840/01842/01844/01850/01852/01860 (elbow/forearm/wrist/hand) → AXILLARY (64417).
   - If the chart only says "brachial plexus block" and the case is 01620/01630/01634/01636/01638 (shoulder/upper arm) → INTERSCALENE (64415).

B. TAP / ABDOMINAL WALL BLOCK FAMILY — laterality must match the surgery:
   - Unilateral abdominal surgery (unilateral inguinal hernia 00830, unilateral nephrectomy, single-side appendectomy, unilateral colectomy at the affected side) → 64486 (unilateral TAP) with LT or RT in M2.
   - Bilateral abdominal surgery (bilateral hernia, midline laparotomy, hysterectomy, c-section, bowel resection across midline) → 64488 (bilateral TAP).
   - If chart says "TAP block" with no laterality, INFER from the surgery: unilateral → 64486, midline/bilateral → 64488.
   - The same logic applies to ESP, paravertebral, rectus sheath: unilateral surgery → unilateral block code; midline/bilateral surgery → bilateral block code.

C. NEURAXIAL FAMILY — site of injection determines the code:
   - Lumbar single-shot, no ultrasound → 62322
   - Lumbar single-shot, WITH ultrasound → 62323
   - Cervical or Thoracic single-shot → 62320
   - Continuous catheter: 62318 (cervical/thoracic) / 62319 (lumbar)

D. LOWER-EXTREMITY NERVE BLOCK FAMILY — pick by what nerve was actually targeted, not by what's most common:
   - Femoral surgery (hip, anterior thigh, knee anterior) → 64447 femoral, 64445 sciatic if posterior knee involved.
   - Adductor canal block for knee surgery is 64447 (same CPT as femoral; described as "adductor canal" or "saphenous").
   - iPACK for posterior knee is 64473.
   - Don't add multiple lower-extremity blocks unless each is independently documented in the chart (e.g. separate "Sciatic" + "Adductor" entries).

E. SINGLE-SHOT vs CATHETER:
   - If the chart says "single shot", "single injection", "one shot" → single-shot CPT.
   - If the chart says "catheter", "continuous", "infusion", "pump" → continuous catheter CPT.
   - If silent → default to single-shot.

F. WHEN THE CHART CONTRADICTS YOUR INFERENCE:
   - Explicit chart wording wins over surgical-site inference. If the chart clearly says "interscalene" for an elbow case, use 64415, but flag mentally — this is unusual.
   - If the chart wording is unclear or just generic ("regional", "block"), use the surgical-site rule.

G. NEVER ADD A BLOCK THAT ANATOMICALLY CANNOT COVER THE SURGICAL SITE:
   - No interscalene for hand surgery.
   - No bilateral TAP for unilateral inguinal hernia.
   - No femoral block for upper-extremity surgery.
   - No iPACK for shoulder.

If after all these checks you still cannot determine the correct subtype, pick the most ANATOMICALLY APPROPRIATE option for the surgical site, not the most common one in the family.

TRANSESOPHAGEAL ECHOCARDIOGRAPHY (TEE)

CRITICAL: TEE is commonly performed during cardiac surgery (CPT 00537, 00567) and MUST be billed when documented. Look carefully for ALL of these indicators in the PDF:
- "Comprehensive TEE" with a timestamp in the Actions/Reportable Actions section
- "TEE Probe Placement" listed under Anesthesia Procedures
- "Indication For TEE:" section with reasons like "Requested By Surgeon" or "Assessment Of Intracardiac/Valvular Pathology"
- CPT codes 93312, 93315, 93320, 93325 listed in the TEE section
- "Comprehensive TEE Post-CPB" (post-cardiopulmonary bypass TEE)
- Action Comments mentioning "Comprehensive TEE @ [time]"

IF TEE was performed (any of the above indicators found), add THREE lines:
1. 93312 — TEE probe/image report (Modifier: 26 in M1, 59 in M2)
2. 93320 — Doppler echo exam (Modifier: 26 in M1, 59 in M2)
3. 93325 — Color flow velocity map (Modifier: 26 in M1, 59 in M2)

All three (93312 + 93320 + 93325) are billed together whenever TEE is performed. Use the SAME provider for all three lines.

⚠️ TEE BILLING RULE: Do NOT bill TEE probe placement only (93318). We only charge when interpretation is also done (93312). Always bill 93312 (interpretation) + 93320 (Doppler) + 93325 (color flow) together when TEE is documented. Never use 93318 alone.

TEE lines are billed IN ADDITION TO vascular lines (36620, 93503, 76937), not instead of them. A cardiac case can have 36620 + 76937 + 93503 + 93312 + 93320 + 93325 all billed together.

SPECIAL RULE: ULTRASOUND GUIDANCE

IF Arterial line or CVP was ULTRASOUND guided: -> add ONE line with code 76937 (Modifier: 26 in M1).

IMPORTANT: 76937 ALWAYS requires modifier 26 in M1. Never bill 76937 without modifier 26.

IMPORTANT: 76937 can only be billed ONCE per encounter, even if multiple lines (A-line + CVP + Swan-Ganz) were all ultrasound guided. Do NOT add multiple 76937 lines.

(The 76937 line goes right after the first vascular line it is indicated for).

FINAL FORMATTING RULES:

If anything is empty, put two semicolons ;; to indicate the section is empty.

NEVER put something in M2 but nothing in M1. It is sequential. 59 is always in M1.

Use same provider mapping strings as given; do not deviate.

If no blocks or lines were done, say: NONE DONE.
