=== peripheral_blocks ===
Description: In this field list out all blocks/lines performed on the patient, using the strict pipe-delimited format below.

OUTPUT FORMAT:
|cpt_code;MD;Resident;CRNA;M1;M2|cpt_code;MD;Resident;CRNA;M1;M2|...

Each block must have EXACTLY 6 semicolon-delimited fields in this order:
1. CPT code
2. MD (always from "Responsible Anesthesiologist")
3. Resident (provider with MD suffix from the "Resident/CRNA" line)
4. CRNA (provider with CRNA suffix from the "Resident/CRNA" line)
5. Modifier 1
6. Modifier 2

If a field is empty, leave it blank between semicolons (e.g. `;;`). NEVER put something in M2 without something in M1 — modifiers are sequential. Modifier 59 ALWAYS goes in M1, NEVER in M2.

Use provider mapping strings exactly as given for the provider fields elsewhere in the template — do not extract provider names directly from the PDF text; map them to one of the canonical provider strings. If you cannot map a provider, leave the field blank rather than guessing.

ONLY output blocks that are documented as ACTUALLY PERFORMED, with a clear dedicated section in the PDF showing: laterality, technique (single-shot vs catheter), provider(s), and time. Do NOT output blocks that are only mentioned in surgical plans, anesthesia plans, schedules, postop summaries, or simple lists like "lines and airways". If no blocks or lines were performed: output `NONE DONE`.

Laterality maps: RT = right, LT = left, 50 = bilateral. For CPT codes that are inherently bilateral (e.g. 64468/64469/64488/64489), do NOT add 50 — the bilaterality is already in the code, just use 59 in M1 with M2 blank.

Continuous catheter wording ALWAYS overrides single-shot.

---

## SCIATIC BLOCK
- Single injection: 64445
- Continuous catheter: 64446
- Modifiers: 59 (M1) + RT (M2), or 59 + LT, or 59 + 50

## INTERSCALENE BLOCK
- Single injection: 64415
- Continuous catheter: 64416
- Modifiers: 59 (M1) + RT (M2), or 59 + LT, or 59 + 50

## AXILLARY BLOCK
- Single injection: 64417
- Continuous catheter: 64418
- Modifiers: 59 (M1) + RT (M2), or 59 + LT, or 59 + 50

## FEMORAL BLOCK
- Single injection: 64447
- Continuous catheter: 64448
- Modifiers: 59 (M1) + RT (M2), or 59 + LT, or 59 + 50

## SAPHENOUS / ADDUCTOR CANAL BLOCK
Saphenous nerve and adductor canal blocks (including in arthroplasty cases) → same codes as femoral:
- Single injection: 64447
- Continuous catheter: 64448
- Modifiers: 59 (M1) + RT or LT

## THORACIC PLANE BLOCKS
Also called: Erector Spinae Plane (ESP), Parasternal block, Serratus anterior plane, Thoracolumbar interfascial plane (TLIP), Transverse Thoracic Plane (TTP), PECS I, PECS II.
- Unilateral single injection: 64466
- Unilateral continuous catheter: 64467
- Bilateral single injection: 64468
- Bilateral continuous catheter: 64469
- 64466/64467 modifiers: 59 (M1) + RT or LT
- 64468/64469 modifiers: 59 only — NO RT/LT/50 (bilaterality is in the code)

## LOWER EXTREMITY FASCIAL PLANE BLOCKS
Also called: Fascia Iliaca block (FIB / FICB / fascia iliaca compartment block), iPACK (interspace between popliteal artery and capsule of knee), PENG (pericapsular nerve group), SPANK (suprainguinal pericapsular).

⚠️ POST-OP PAIN REQUIREMENT (BLOCKS ONLY — does NOT apply to vascular lines, TEE, or ultrasound guidance):

Peripheral nerve blocks, spinal blocks, and epidural blocks are ONLY billable as second-line charges when they are performed SPECIFICALLY for POST-OPERATIVE PAIN management — separate from the primary anesthetic.

DO NOT bill a block (or include it in this field) if:

1. The block IS the primary anesthetic for the case. For example, if "Spinal" is documented as "Primary anesthetic" / "Indication: Primary anesthetic" / "Requested by surgeon for primary anesthesia", it is the anesthesia for the case itself — NOT a post-op pain block. Do not add it as a peripheral_block line.

2. There is no documentation that the block was for post-operative pain. Look for explicit indicators like: "for post-op pain", "post-operative pain control", "post-op analgesia", "requested by surgeon for post-op pain", or a clearly separate block performed before/after the case for analgesic purposes.

3. The indication is "intraoperative anesthesia" or the block is listed under the primary anesthesia type (e.g., "Anesthesia Type: Spinal" with no separate post-op block documented).
